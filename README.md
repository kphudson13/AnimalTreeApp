# **Animal Tree of Life**  
An interactive tree for higher animal clades. 
This web app is designed to be a learning tool for invertebrate diversity. 
You are welcome to use and modify the app (see [LICENSE](https://github.com/kphudson13/AnimalTreeApp/blob/main/LICENSE)).   
The app is built in R to be deployed on shiny servers. You could run the app locally as well, or deploy to a server yourself.   
Visit the site [here](https://kphudson.shinyapps.io/AnimalTreeApp/)  
>[!NOTE]
>How some clades are organized is contentious amoung taxonomists, take this for what it is.  
>Some support code will not work unless you have an OTL API key.
